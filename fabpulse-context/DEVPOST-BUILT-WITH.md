# FabPulse — Built With
# Devpost "Built with" section

---

## Languages
- Python 3.12
- TypeScript
- SQL

## Frameworks & Libraries
- FastAPI — backend API framework
- React 18 — frontend (dashboard + kiosk)
- Vite — frontend build tool
- Tailwind CSS — styling
- Workbox — service worker / offline PWA
- SQLAlchemy — ORM
- Alembic — database migrations
- TanStack Query — data fetching and caching
- Zustand — frontend state management
- Recharts — data visualization
- idb — IndexedDB wrapper for offline queue

## Cloud Platform
- Google Cloud Run — backend hosting (serverless containers)
- Google Vertex AI — Gemini 1.5 Pro (SEC Simulator + generative reports)
- Google Firebase — PWA hosting + offline sync bridge
- Google BigQuery — analytics warehouse + AI execution logs
- Google Cloud Logging — agent execution audit trail
- Google Cloud Tasks — async PDF report generation
- Google Artifact Registry — Docker image registry

## Database
- PostgreSQL (via Supabase) — primary relational database
- Supabase Realtime — WebSocket live updates to dashboard
- Supabase Auth — JWT authentication + Row Level Security
- IndexedDB — local offline queue on kiosk devices

## APIs & Services
- Vertex AI API (Gemini 1.5 Pro) — Monte Carlo simulation + narrative report generation
- Stripe API — subscription billing (Starter / Growth / Pro tiers)
- Stripe Webhooks — automated plant provisioning on payment
- Background Sync API — automatic offline data flush on reconnect
- Fullscreen API — kiosk lockdown mode on workstation tablets

## DevOps & Infrastructure
- Docker — containerization
- GitHub Actions — CI/CD pipeline
- Google Cloud Build — container builds
