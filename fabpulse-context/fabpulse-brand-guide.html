Do's & Don'ts
✓ Do
+ Use Inter at 700 for headlines
+ Use teal only for primary actions and brand moments
+ Use red exclusively for alerts and errors
+ Keep icon on a teal rounded square
+ "Fab" dark / neutral, "Pulse" always teal
+ Dark background (#0D1117) for hero sections
✗ Don't
− Use the logo without the icon mark
− Use all-gray or unstyled wordmark
− Add gradients or glow effects to the logo
− Use more than 3 colors in one layout
− Use red for anything other than errors
− Use light weight (400) for the wordmark
